## Answer to the Question 1 (a)
```
(i) 1UIU : invalid (starting with a digit)
(ii) SPL_2023 : valid
(iii) char : invalid (it's a keyword)
(iv) SPL$ : invalid (there have a special character $)
(v) My-Course : invalid (there have a special character -)
```
