## Answer to the Question 1 (b)
```
_Num1 : valid
1time : invalid (starting with a digit)
$VALUE : invalid (there have a special character $)
“myValue” : invalid (there have a special character "")
num6 : valid
first_name : valid
last-name : invalid (there have a special character -)
f1x : valid
```
